IOT-based Smart Irrigation System

NodeMCU ESP32 Flutter Firebase C++

Welcome to IoT-powered smart irrigation system! I designed this hardware and software solution to transform small urban gardens into self-sustaining, eco-friendly green spaces. On the physical side, Me and my team engineered a custom sensor network using a NodeMCU ESP32S microcontroller , integrating soil moisture, temperature, and air quality sensors. To bring this data to the user's fingertips, I developed a sleek mobile application using Flutter , providing an intuitive dashboard to monitor real-time garden health and track water usage from anywhere.

The system relies on embedded programming to process real-time environmental data. It acts intelligently, automatically triggering watering via solenoid valves and diaphragm pumps, a method that can cut water waste by up to 50%. The architecture is bridged by Firebase, handling cloud data storage and delivering instant push notifications for critical events like low soil moisture. Complete with a touch-sensor for manual overrides and robust power regulation circuitry , this project highlights my ability to seamlessly merge embedded hardware engineering with modern mobile app development.
